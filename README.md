# БЦ «Двинцев» — Учёт парковки

Веб-приложение для учёта парковочных мест в бизнес-центре «Двинцев» (Klaus Multiparking).

## Возможности
- Учёт парковочных ячеек по блокам (А / В / С / Наземная / Вне ячеек)
- Три режима просмотра: Схема паркинга (PDF), Блоки, Наземная
- Фильтрация, поиск, модалки редактирования
- Две темы: тёмная (оранжево-чёрная) и «Тёплая бумага» (светлая)
- Live-часы, статистика занятости в шапке

## Запуск
Открой `parking.html` в браузере. Всё работает офлайн (single-file приложение).

Для корректной загрузки `scheme.pdf` во вкладке «Схема» запусти локальный сервер:
```bash
python3 -m http.server 8000
# открой http://localhost:8000/parking.html
```

## Файлы
- `parking.html` — основное приложение
- `scheme.pdf` — схема паркинга
- `scheme.b64.txt` — base64-версия PDF (fallback для песочниц без file://)

## Технологии
Чистый HTML/CSS/JS, без сборки. Шрифты: Bebas Neue, Space Grotesk, JetBrains Mono (Google Fonts).
